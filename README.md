# Webpack for Static HTML Websites

## Features
- CSS
- Scss
- Stylus
- Autoprefix
- Auto Reload
- Use `require`
- Use `module.exports`
- ES6 syntax
- Remove `console.log` for production build

## Installation
`npm install`
`npm install webpack-dev-server -g`

## Development
`npm run server`

## Production Build
`npm run prod`
